ConfigCacheBundle
=================

ConfigCacheBundleは設定ファイルキャッシュを扱うバンドルです。  
バンドルごとに定義された設定ファイルをキャッシュします。  
また設定ファイルキャッシュをオブジェクトとして持ちSymfony上でサービス化することで扱いやすい形にします。
